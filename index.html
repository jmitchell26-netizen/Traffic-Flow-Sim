Getting Started

1

Clone the Repository

Get the source code from GitHub

git clone https://github.com/jmitchell26-netizen/Traffic-Flow-Sim.git
2

Get a TomTom API Key

Sign up for a free API key at developer.tomtom.com

3

Configure Environment

Copy the example env file and add your API key

cd traffic-flow-sim/backend
cp ../env.example .env
# Edit .env and add your TOMTOM_API_KEY
4

Run the Application

Use the convenient startup script

cd traffic-flow-sim
./start.sh

Or start backend and frontend separately using the individual scripts.
